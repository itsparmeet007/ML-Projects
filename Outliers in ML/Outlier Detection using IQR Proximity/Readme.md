# 📊 Outlier Detection Using the IQR Method

## 📌 Overview

This project demonstrates how to detect and remove outliers using the **Interquartile Range (IQR)** technique in Python. The notebook performs exploratory data analysis (EDA), identifies skewed distributions, visualizes outliers, and applies the IQR rule to detect extreme values.

This project is ideal for beginners in **Data Science**, **Machine Learning**, and **Data Preprocessing**.

---

## 📂 Dataset

The notebook uses a placement dataset containing features such as:

- CGPA
- Placement Exam Marks

The focus is on analyzing feature distributions and detecting outliers.

---

## 🚀 Project Workflow

1. Import required libraries
2. Load the dataset
3. Perform Exploratory Data Analysis (EDA)
4. Check data distribution
5. Calculate skewness
6. Visualize data using:
   - Distribution plots
   - Box plots
7. Compute:
   - Q1 (25th Percentile)
   - Q3 (75th Percentile)
   - IQR (Interquartile Range)
8. Calculate lower and upper bounds
9. Detect outliers
10. Remove or cap outliers for further analysis
